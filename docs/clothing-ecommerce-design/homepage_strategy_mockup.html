Layout Strategy Zones

Zone 1

Asymmetric Editorial Hero

Showcases our premium quality and design at the absolute top. Splits real estate 50/50 to pair high-impact design photography with a monospace garment diagnostics technical sheet.

Zone 4

Narrative Capsule Tabs

Instead of generic category links, users filter by high-concept collections (e.g. [ 01 / TECH GRAPHICS ] or [ 02 / RETRO NOSTALGIA ]) which elevates the apparel catalog to high art.

Zone 2

Active Drips Grid

Displays only the 3 or 4 live designs currently available for printing. Employs the Playbook rules for 3:4 portrait ratios and immediate size availability chips on hover.

Zone 3

Vault Archive Teaser

The ultimate FOMO generator! Shows recently sold out designs in grayscale with a diagonal strike-through, proving that designs drop, sell out, and are vaulted forever.

How to Code This

This proposal organizes our catalog into structured lists inside the code, utilizing the status flags to handle active vs vaulted display states automatically.

ProductList.jsx Reference
// Proposed structure filter
const activeDrips = PRODUCTS
  .filter(p => !p.isVaulted)
  .slice(0, 3);

const recentVault = PRODUCTS
  .filter(p => p.isVaulted)
  .slice(0, 3);
